# Система сверки финансовых транзакций

## Описание проекта

Это тестовое техническое задание. Проект представляет собой автоматизированную систему для сверки финансовых транзакций из двух различных источников: `RpaBank` и `Pindodo`. Система парсит данные из предоставленных отчетов, выполняет их сравнение на основе заданных ключей (дата, сумма, валюта, номер карты, ID терминала) и генерирует подробный Excel-отчет, который включает:

*   Успешно совпавшие транзакции.
*   Транзакции, найденные только в отчете `RpaBank`.
*   Транзакции, найденные только в отчете `Pindodo`.

## Установка

1.  **Клонируйте репозиторий:**
    ```bash
    git clone <https://github.com/hydrauluu/test_tech_task.git>
    cd test_tech_task
    ```

2.  **Создайте и активируйте виртуальное окружение:**
    ```bash
    python -m venv venv
    # Активация (Windows):
    # .\venv\Scripts\activate
    # Активация (macOS/Linux):
    # source venv/bin/activate
    ```

3.  **Установите зависимости:**
    ```bash
    pip install -r requirements.txt
    ```

4.  **Подготовьте входные файлы:**
    Убедитесь, что файлы `RpaBank_report.txt` и `Pindodo_report.txt` находятся в соответствующих папках (`RPA/` и `PINDODO/`).

## Использование

Запустите основной скрипт из корневой директории проекта:

```bash
python script/main.py
```

Результаты сверки будут сохранены в файле `Результат/reconciliation.xlsx`. Логи выполнения будут доступны в консоли и в файле `logs/reconciliation.log` (если эта директория и файл будут созданы).
